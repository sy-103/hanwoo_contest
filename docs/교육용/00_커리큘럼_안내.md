# 한우 프로젝트 — 데이터 분석 교육 커리큘럼

> 이 시리즈는 **데이터 분석을 처음 시작하는 팀원**이 혼자 읽고도 이해할 수 있도록 만든 학습용 강의안입니다.
> 기존 `19_분류모델_최종.md` / `20_회귀모델_최종.md`는 **"바로 돌리는 실습 코드 모음"**이고, 이 `교육용/` 폴더는 **"왜 그렇게 하는지를 처음부터 배우는 교과서"**입니다. 두 개를 같이 보세요.

---

## 이 커리큘럼을 만든 이유

대회 결과물을 내는 것도 목적이지만, **이 프로젝트로 데이터 분석을 제대로 공부하는 것**이 더 큰 목적입니다. 그래서 "우리가 쓰는 기법"만 딱 설명하지 않고, **그 기법이 속한 분야 전체**를 넓게 다룹니다. 예를 들어 "타깃 인코딩"을 배울 때 타깃 인코딩만 보지 않고, **인코딩이라는 것 자체가 무엇이고 어떤 종류가 있으며 각각 언제 쓰는지**부터 시작합니다.

각 강의는 다음 원칙으로 썼습니다.
- **비유 먼저, 공식 나중**: 추상적인 수식 전에 일상 비유로 직관을 잡습니다.
- **"왜?"에 답한다**: "이렇게 하세요"가 아니라 "왜 이렇게 해야 하는지"를 설명합니다.
- **한 번에 하나씩**: 개념을 작게 쪼개 단계적으로 쌓습니다.
- **끝에 요약과 자가점검**: 읽고 나서 머릿속에 남도록 핵심 정리와 스스로 답해볼 질문을 둡니다.
