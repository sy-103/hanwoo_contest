회귀팀은 추가 설치가 거의 없습니다(분류팀과 같은 환경이면 OK).

핵심 도구 두 개:
- **statsmodels**: 우리 메인 해석 도구. p값·신뢰구간·강건 표준오차 등 **통계 정보가 풍부**합니다. (sklearn의 LinearRegression은 예측만 주고 p값이 없어서, 해석엔 statsmodels를 씁니다.)
- **scikit-learn**: 규제 회귀(Lasso/Ridge/ElasticNet)와 교차검증용.

## 3. 꼭 이해하고 넘어갈 셀 4개

### 셀 4 — 변수 설계: 교란/매개/관심을 코드로 가른다

[R1](R1_회귀의_목적_인과.md)의 핵심이 여기서 코드가 됩니다.

```python
weather_vars = ["ratio_고온", "ta_min_mean", ...]      # (A) 관심 변수
confounders  = ["AGE", "sido_*", "계절더미", "연도더미", ...]  # (B) 교란 → 넣는다
mediators    = ["WEIGHT", "daily_gain"]                # (C) 매개 → 총효과에선 뺀다!

reg_features        = weather_vars + confounders              # 총효과 모델(메인)
reg_features_direct = weather_vars + confounders + mediators  # 직접효과 모델(보조)
```

**이 구분이 회귀 분석의 성패를 가릅니다.** 몸무게·증체를 confounders에 잘못 넣으면([R1](R1_회귀의_목적_인과.md)) 더위 효과가 22% 잘립니다. 주석으로 "왜 빼는지/넣는지"를 남겨 둡니다.

### 셀 6 — VIF 반복 제거

[R3](R3_다중공선성_VIF.md)의 VIF를 구현. 가장 높은 변수부터 빼고 다시 계산을 반복합니다.

```python
while True:
    vif = fast_vif(X)              # 상관행렬 역행렬 대각 (빠른 VIF)
    if vif.max() <= 10: break      # 다 10 미만이면 멈춤
    X = X.drop(가장_높은_변수)      # 예: AGE↔사육일수 중 하나
```

**읽는 법**: 제거 로그에 무엇이 나오는지 보세요. 월령↔사육일수처럼 "정의상 같은 변수"가 걸리면 정상. 관심 변수(더위)가 걸리면 어떤 통제 변수와 얽혔는지 13회차 상관표와 대조.

### 셀 7~8 — OLS(HC1)와 계수표

[R2](R2_선형회귀_기초.md)의 OLS + [R5](R5_계수해석_한계효과.md)의 해석을 구현.

```python
ols = sm.OLS(y, X).fit(cov_type="HC1")     # 이분산 강건 표준오차
coef["효과%_1단위"] = (np.exp(b) - 1) * 100        # 정확식 (R5)
coef["효과%_1SD"]  = (np.exp(b * 표준편차) - 1) * 100  # 영향력 비교
```

근사(b×100)가 아니라 **정확식 (e^b−1)×100%** 를 쓰는 게 핵심([R5](R5_계수해석_한계효과.md)). 1단위 효과와 1-SD 효과를 둘 다 뽑습니다.

### 셀 10.5 — 총효과 vs 직접효과

[R1](R1_회귀의_목적_인과.md)·[R5](R5_계수해석_한계효과.md)의 매개 분해를 구현. 매개 포함/제외 두 모델을 돌려 더위 계수를 비교합니다.

```python
b_total  = ols.params["ratio_고온"]          # 매개 제외 (총효과)
b_direct = ols_direct.params["ratio_고온"]   # 매개 포함 (직접효과)
간접효과 = b_total - b_direct                 # 체격을 통한 경로
```

**보고서엔 총효과가 헤드라인**, 차이는 부연.

## 4. 처음 돌릴 때 자주 만나는 함정

| 증상 | 원인 | 해결 |
| --- | --- | --- |
| `exog contains inf or nans` | 무한대/결측이 OLS에 들어감 | `prepare_X`로 inf→NaN→중앙값 (셀 5) |
| VIF 계산이 SVD 에러 | 분산 0인 상수 컬럼 | `fast_vif`가 분산 0 컬럼 자동 제외 |
| 계수 부호가 도메인과 반대 | 다중공선성([R3](R3_다중공선성_VIF.md)) | 그 변수 VIF 확인, 높으면 해석 보류 |
| 더위 효과가 너무 작음(거의 0) | 매개변수를 통제함([R1](R1_회귀의_목적_인과.md)) | 몸무게·증체를 총효과 모델에서 제외 |
| 모든 변수가 p<0.001 | 표본 122만 | p값 말고 효과 크기·CI로 해석([R5](R5_계수해석_한계효과.md)) |

> **가장 흔한 개념 실수**: 셀 4에서 WEIGHT·daily_gain을 "교란"으로 넣는 것. [R1](R1_회귀의_목적_인과.md)을 다시 읽으세요 — 이건 매개라 총효과 모델에선 빠져야 합니다.

6. 셀 12~13: 규제 3종으로 강건성·변수 선택 확인.
